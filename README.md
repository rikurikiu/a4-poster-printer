# A4 分割ポスター印刷アプリ

1枚の画像を **A1 などの大判サイズに拡大** し、**A4 用紙に分割して印刷**、
切って貼り合わせると 1 枚の大きなポスターになる、インストール不要のブラウザアプリです。

催事・店頭ポップ・告知ポスターなどを、家庭やオフィスの A4 プリンターで手軽に作れます。

## 使い方

1. `index.html` をブラウザ（Chrome / Safari / Edge など）で開く
2. **画像を選択**（クリックで選ぶか、画面にドラッグ＆ドロップ）
3. 「組み上がりイメージ」で割り付けを確認
   （青い線＝用紙の境目、数字＝印刷順）
4. **🖨 印刷する** を押す
5. 印刷ダイアログで設定：
   - 用紙＝**A4** / 余白＝**なし** / 倍率＝**100%**（「用紙に合わせる」はOFF）
   - **「背景のグラフィック」をON**
6. 印刷後、各紙の **赤い切り取り線** で切り、**のりしろ** を重ねて貼り合わせる

> PDF 保存も同じ手順でできます。

## 調整できること

- **完成サイズ**：A1（既定）／ A0 ／ A2　＋ 縦 / 横の切り替え
- **分割数（列 × 行）**：手動変更も可（サイズ変更時は必要枚数を自動提案）
- **のりしろ**：0〜15mm（貼り合わせの重ね幅。赤線が仕上がり位置）
- **画像の収め方**：枠いっぱい（推奨）／ 全体を表示 ／ 変形
- **切り取り線・ページ番号** の印刷 ON/OFF（各紙に「列◯行◯」を表示）

## 特長

- インストール不要。`index.html` を開くだけ
- 画像はブラウザ内（あなたの PC 内）だけで処理され、外部に送信されません
- 必要なファイルは `index.html` 1 つだけ
